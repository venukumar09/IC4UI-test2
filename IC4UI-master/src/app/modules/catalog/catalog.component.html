Catalog Guide

Connector

Configure reusable and secure connection with the data source. Use the connection for data and metadata extraction. Supported connections - Database. File system, cloud.

Register

Registration is the process of defining the source data.Describe the data types and formats. Register the source file as an object to the data lake and access it with the registered object name for data ingestion and consumption.

Business Glossary

Build and manage a common business vocabulary and make it available across an organization. this vocabulary furnishes clear meaning and business context and can be linked to the underlying technical metadata to provide a direct association between business terms and objects.
